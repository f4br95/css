Realizar uma pesquisa eleitoral com todos os eleitores está fora das possibilidades, tanto por uma questão de disposição de recursos, bem como a possibilidade de recusa por parte dos indivíduos a serem consultados. Nesse sentido, optando por não ouvir todos os eleitores, seria interessante consultar 383 pessoas. Para realizar esta pesquisa, poderia ser usada a estratégia de amostragem sistemática, escolhendo um eleitor a cada número de eleitores, até ouvir toda a quantidade definida. O resultado da pesquisa seria justificado pela margem de erro apresentada, que com 383 pessoas seria de 5%.
